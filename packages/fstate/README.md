# Contributions

## Run test

### Install dependencies for tests

```bash
dart pub global activate coverage
dart pub global activate full_coverage
```

and install `lcov` and `genhtml` for coverage report.

### Run tests

Each package has `scripts/run_test.sh` to run tests.
